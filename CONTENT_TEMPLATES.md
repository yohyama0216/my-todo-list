# コンテンツテンプレート集

このドキュメントでは、デザイン＆開発サービスのポートフォリオサイトで使用できるテキストテンプレートを提供します。
実際の内容に合わせてカスタマイズしてご使用ください。

## 🎯 サイト構成概要

- **トップページ (`/`)**: デザインと開発、2つのサービス入口を明確に提示
- **デザインページ (`/design`)**: チラシ・ポスター制作の実績と料金
- **開発ページ (`/dev`)**: PHPシステム開発の実績と料金
- **実績一覧 (`/works`)**: カテゴリフィルタ付き全実績
- **料金 (`/pricing`)**: デザイン・開発の料金体系
- **お問い合わせ (`/contact`)**: サービス種別選択付きフォーム

## 📝 ホームページ

### キャッチコピー例

```
1. デザインと開発で ビジネスを加速する
2. チラシ制作からシステム開発まで 一貫サポート
3. 発注者目線で分かりやすい デザイン＆開発サービス
4. あなたのビジネスに最適な デザイン・開発ソリューション
5. 実績と成果で選ばれる フリーランスのデザイン＆開発
```

### サービス入口の説明文

```
【デザインサービス】
店舗、イベント、広告など、目的に応じた印象的なデザインを制作。
ターゲットに響くビジュアルで、集客・認知度向上をサポートします。

【開発サービス】
既存システムの改修・保守から新規開発まで対応。
業務効率化や管理画面構築で、ビジネスの成長を技術面から支援します。
```
