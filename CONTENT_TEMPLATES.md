# コンテンツテンプレート集

このドキュメントでは、開発サービスのポートフォリオサイトで使用できるテキストテンプレートを提供します。
実際の内容に合わせてカスタマイズしてご使用ください。

## 🎯 サイト構成概要

- **トップページ (`/`)**: サービス入口を明確に提示
- **開発ページ (`/dev`)**: PHPシステム開発の実績と料金
- **クロスワードページ (`/crossword`)**: クロスワードパズル制作の実績と料金

---

## 📝 ホームページ

### キャッチコピー例

```
1. システム開発で ビジネスを加速する
2. システム開発から一貫サポート
3. 発注者目線で分かりやすい 開発サービス
4. あなたのビジネスに最適な 開発ソリューション
5. 実績と成果で選ばれる フリーランスの開発
```

### サービス入口の説明文

```
【開発サービス】
既存システムの改修・保守から新規開発まで対応。
業務効率化や管理画面構築で、ビジネスの成長を技術面から支援します。

【クロスワードパズル制作サービス】
オリジナルのクロスワードパズルを制作します。
雑誌、イベント、教育用など、様々な用途に対応いたします。
```

### 使用技術
PHP 8.1、MySQL 8.0、Redis、Git など
